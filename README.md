# Retail-Analysis
Analisis ini dibuat untuk memenuhi tugas akhir dari Bootcamp yang saya ikuti sebagai Data Analyst di Practicum. Disini saya bertindak sebagai Junior Data Analyst yang melakukan analysis mengenai retail.
Sebelum melalukan analisis saya diharuskan untuk menyusun deskripsi bagaimana tahapan perumusahan dalam penyelesaian project dan mengkalrifikasi detail project kepada Team Leader (dalam hal ini Tutor akan menjadi Team Lead).

Berikut tahapan project yang saya lakukan:
Tahapan proyek:
* 		Pra-pemrosesan data
* 		Analisis data eksploratif
* 		Pengujian hipotesis
* 		Penggunaan metrik dan indikator bisnis
* 		Persiapan presentasi
